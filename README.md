# AI Mental Health Chatbot

An AI-powered chatbot that helps users understand their emotional state through sentiment analysis and provides supportive responses.

## Features

* Emotion Detection
* Mental Wellness Suggestions
* User-Friendly Interface
* Built with Flask and Python

## Tech Stack

* Python
* Flask
* TextBlob
* HTML/CSS

## Installation

```bash
pip install flask textblob
python app.py
```

## Usage

Open:

```text
http://127.0.0.1:5000
```

Enter your thoughts and the chatbot will analyze your emotion and provide a response.
